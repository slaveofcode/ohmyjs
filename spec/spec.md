## Features

* Define parser component
* Various Types
* Key-Value Map
* Extended

* Check instance of component
* Multi-level parser
* Valid value based on ENUM
* Configurable component validator (default is inactive)
* immutable results
* custom value mapping
* Options to allow unknown keys to be shown
